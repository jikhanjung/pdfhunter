# Goals

## 핵심 목표

> **왜 이 서지 정보가 선택되었는지 설명 가능한 자동 추출**

## 구체적 목표

1. **다중 소스 추출**: 규칙 기반(regex), Vision LLM, 텍스트 LLM, 외부 서지 DB(Crossref)를 결합하여 서지 정보를 추출한다.
2. **Evidence 중심 설계**: 모든 추출 필드는 근거(페이지 번호, 원문 텍스트, 이미지 영역)와 함께 저장한다.
3. **Confidence 기반 라우팅**: 신뢰도 점수에 따라 자동 확정(confirmed), 사용자 검수(needs_review), 실패(failed)로 분류한다.
4. **다국어 OCR 지원**: 영어, 프랑스어, 러시아어 문서를 안정적으로 처리한다.
5. **에이전트 루프**: 누락 필드가 있을 때 추가 페이지 탐색, 웹 검색 등을 자동으로 시도한다(반복 횟수 제한 포함).
6. **사용자 검수 UI**: 추출 결과와 근거를 함께 보여주고, 승인/수정/재시도를 지원한다.
7. **다중 출력 포맷**: CSL-JSON(기본), RIS, BibTeX로 내보내기를 지원한다.
8. **대량 처리 품질 목표** (10,000 PDF 기준):
   - confirmed_auto: 85–95%
   - needs_review: 5–15%
   - failed: < 3%
